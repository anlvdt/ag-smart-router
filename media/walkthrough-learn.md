# AI Learning Engine

Grav learns from your workflow using adaptive learning (SGD + momentum):

- **Tracking** — Observes terminal commands you approve or reject
- **Epochs** — Training cycles that refine command safety predictions
- **Promotion** — Automatically suggests commands for the whitelist after enough approvals
- **Wiki** — Builds a knowledge base of command patterns and concepts

View learning data with **Grav: View Learning Stats** or check the Dashboard's Session metrics.
