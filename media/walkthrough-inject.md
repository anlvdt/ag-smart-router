# Inject Runtime

Grav injects a lightweight runtime into Antigravity IDE to enable:

- **Auto-Click** — Automatically clicks approval buttons (Run, Accept, Approve)
- **Auto-Scroll** — Keeps the chat panel scrolled to the bottom

The runtime is injected on activation and updates automatically when Grav is updated.

> Tip: Use `Ctrl+Shift+D` to open the Dashboard and toggle features on/off.
