# Inject Runtime

Grav works by injecting a lightweight runtime script into Antigravity's workbench.

**What it does:**
- Auto-clicks Run, Allow, Accept, Continue buttons
- Keeps chat scrolled to bottom
- Detects quota exhaustion (25+ phrases)
- Suppresses corrupt/reinstall banners

The runtime is automatically injected on first activation. You can manually re-inject with `Grav: Inject Runtime`.
