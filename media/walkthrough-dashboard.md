# Dashboard

The Grav Dashboard is your control center:

- **Controls** — Toggle Auto-Click, Auto-Scroll, and Dry Run mode
- **Timing** — Adjust click interval, scroll interval, and pause duration
- **Patterns** — Enable/disable which button labels to auto-click
- **Session** — Real-time metrics (clicks, uptime, messages, learning epoch)
- **Brain** — View whitelist/blacklist counts and wiki stats
- **Activity** — Live feed of clicks, approvals, and blocked commands

Open with `Ctrl+Shift+D` or via the command palette: **Grav: Dashboard**
